Fetch API

Take me to home

resource: https://developer.mozilla.org/en-US/docs/Web/API/Fetch_API

  • an 'interface' for fetching resources
  • more modern than XMLHttpRequest
  • one mandatory argument: path to the resource to be fetched
  • returns a promise that resolves to the response (successful or not)
  • optionally pass an init options object as the 2nd arg

diffs from jquery

  • promise returned from fetch won't reject on http error status
  • will only reject on network failure or if anything prevented the request from completing
  • something something cookies...

fetch interfaces

  • WindowOrWorkerGlobalScope.fetch() => to fetch a resource
  • Headers => response/request headers. query and take diff actions per the results
  • Request => of a resource
  • Response => to a request

fetch mixin

  • Body => rel'd to the response/request. declare content type and handling instructions

browser support

  • chrome, edge, firefox, opera, safari... everyone but:
  • internet explorer
